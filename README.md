# Task Manager

A Trello-inspired Kanban Task Manager built using Vanilla JavaScript.

## Features

- ✅ Create Tasks
- ✅ Edit Tasks
- ✅ Delete Tasks
- ✅ Drag & Drop
- ✅ Local Storage
- ✅ Dark Mode
- ✅ Keyboard Shortcuts
- ✅ Responsive Design

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
